# AstroLog 安装部署指南

> 版本：v1.0 | 适用：AstroLog 天文观测日志与星体管理系统

---

## 1. 环境要求

| 组件     | 最低版本                | 说明                               |
| -------- | ----------------------- | ---------------------------------- |
| JDK      | 17+                     | Java 运行环境，推荐 OpenJDK 17 LTS |
| MySQL    | 8.0+                    | 数据库服务器，需启用 InnoDB 引擎   |
| Maven    | 3.8+                    | 构建工具（仅源码编译时需要）       |
| 操作系统 | Windows / macOS / Linux | 跨平台 Java 桌面应用               |

---

## 2. 数据库初始化

### 2.1 创建数据库

登录 MySQL 并创建数据库：

```sql
CREATE DATABASE astrolog DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2.2 执行初始化脚本

```bash
mysql -u root -p astrolog < sql/init.sql
```

init.sql 将创建以下 11 张表：

- `users` — 用户账户
- `celestial_bodies` — 星体数据
- `observation_sites` — 观测地点
- `equipment` — 器材信息
- `observations` — 观测记录
- `obs_equipment` — 观测-器材关联
- `equipment_maintenance` — 器材维护日志
- `observation_tags` — 标签定义
- `obs_tag_relation` — 观测-标签关联
- `user_favorites` — 用户收藏
- `operation_logs` — 操作日志

---

## 3. 配置文件

编辑 `src/main/resources/db.properties`（源码运行）或 classpath 下的 `db.properties`（JAR 运行）：

```properties
db.url=jdbc:mysql://localhost:3306/astrolog?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=Asia/Shanghai
db.username=root
db.password=your_password
db.maxConnections=10
```

| 参数                | 说明                          |
| ------------------- | ----------------------------- |
| `db.url`            | 数据库连接 URL，默认端口 3306 |
| `db.username`       | MySQL 用户名                  |
| `db.password`       | MySQL 密码                    |
| `db.maxConnections` | 连接池最大连接数，默认 10     |

---

## 4. 编译打包

### 4.1 源码编译

```bash
git clone <repository-url>
cd AstroLog-Design
mvn clean compile
```

### 4.2 运行测试

```bash
mvn test
```

预期：135 个测试用例全部通过。

### 4.3 生成 Fat JAR

```bash
mvn clean package
```

生成的 JAR 文件位于 `target/astrolog-1.0-SNAPSHOT-jar-with-dependencies.jar`，包含所有依赖。

---

## 5. 启动运行

### 5.1 Fat JAR 启动（推荐）

```bash
java -jar target/astrolog-1.0-SNAPSHOT-jar-with-dependencies.jar
```

### 5.2 Maven 启动（开发模式）

```bash
mvn exec:java -Dexec.mainClass="com.astrolog.AppMain"
```

### 5.3 启动验证

- 应用窗口正常弹出（全屏模式，最小 1024×700）
- 登录界面出现
- 注册新用户后可以登录并进入主界面

---

## 6. 数据备份恢复

### 备份

在系统设置面板 → 数据管理 → 点击「备份数据」→ 选择保存目录 → 点击「开始备份」。
生成 `astrolog_backup_yyyyMMdd_HHmmss.sql` 文件。

### 恢复

在系统设置面板 → 数据管理 → 点击「恢复数据」→ 选择 .sql 备份文件 → 确认严重警告 → 点击「开始恢复」。

> **警告：** 恢复操作将覆盖当前数据库全部数据，请确认已备份当前数据。
