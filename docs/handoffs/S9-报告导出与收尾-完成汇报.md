## 6. 已知限制

1. **BackupService.restore 内存模式：** 使用 `Files.readString()` 全部加载到内存 + `split(";\\s*\n")` 分割语句。大数据量备份可能 OOM；含分号的文本字段可能分割错误。交接文档已标注此限制，建议未来改为流式解析。

2. **JaCoCo 覆盖率 20.7%：** 因无 MySQL 运行环境，DAO 层集成测试无法执行。单元测试覆盖 Service 层全部业务逻辑。

3. **JasperReports PDF 中文：** 依赖 `itext-asian:5.2.0` 的 STSong-Light 字体。实际环境如缺少该字体可能显示方框。
