## 4. S5 开发偏差记录

| 偏差                                | 原因                             | 影响                                            |
| ----------------------------------- | -------------------------------- | ----------------------------------------------- |
| pom.xml `@{argLine}` → `${argLine}` | Windows cmd.exe 不兼容 `@` 字符  | JaCoCo prepare-agent 设置 `${argLine}` 正常工作 |
| SiteDao update/delete 自行实现      | Handoff 留空                     | 按表结构补充完整                                |
| AddObsDialog 星体搜索用 JPopupMenu  | Handoff 未详规                   | 功能可用，无键盘导航                            |
| 器材选择 JList → JCheckBox          | JList invokeLater 切换方案不可靠 | 复选框视觉反馈明确，操作直观                    |
| ObsService 移除 siteDao 依赖        | Handoff 规格包含但未实际使用     | 死代码清理                                      |
| 筛选栏 3 行布局                     | 时间分区后控件增多致换行遮挡     | GridLayout(3,1) 确保所有控件可见                |
