## 4. S6 开发偏差记录

| 偏差                                                    | 原因                                                               | 影响                                            |
| ------------------------------------------------------- | ------------------------------------------------------------------ | ----------------------------------------------- |
| StatsService 新增 `observedBodyIds()` 方法              | StarMapCanvas 需要已观测星体 ID 集合筛选星图，handoff 未包含此方法 | 11 个方法（交接估算 10 个）                     |
| `topEquipment` 使用 `batchGetUsageCounts` 批量查询      | S4 偏差经验要求避免 N+1，EquipDao 已有批量方法                     | 性能优于 handoff 的逐条查询方案                 |
| `totalEquipmentUsed` 仍逐条调用 `getLinkedEquipmentIds` | ObsDao 无批量获取关联器材的方法，<5000 条可接受                    | N+1 查询，后续可补充 batchGetLinkedEquipmentIds |
| StatsPanel 额外持有 `BodyDao` 字段                      | StarMapCanvas 需要加载全部星体数据                                 | 轻微偏离三层架构纯粹性                          |
