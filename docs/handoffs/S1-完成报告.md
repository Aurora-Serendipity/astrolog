## 6. 已知问题

1. **JaCoCo + Java 25 兼容性：** JaCoCo 0.8.11 的 ASM 不支持 Java 25 class file version 69，`mvn test` 时有 ASM 警告但测试正常执行。如需覆盖率报告，建议升级 JaCoCo 到 0.8.12+。
2. **Validator 包级可见性：** Validator/ValidationResult 目前为 package-private，S2 若 Service 层需使用，需提升为 public 并拆分文件。
3. **BaseDao.dataExists 表名拼接：** 使用字符串拼接表名（参数来自 DAO 子类硬编码），Semgrep 报告形式警告，实际无注入风险。
