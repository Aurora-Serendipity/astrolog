## 3. 与交接文档的偏差说明

| 偏差项                                      | 说明                                                                                                                                                 |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **BodyServiceTest 用例数 14 > 10**          | 追加 BS-011~014：isFavorited、getFavorites、listByPopularity、十进制坐标解析，覆盖更完整                                                             |
| **FavoriteDao.exists() 实现修正**           | 原设计用 `executeQuery(COUNT(*))` + `!results.isEmpty()`，COUNT 始终返回一行导致始终 true。改为原生 JDBC 检查 count 值 > 0                           |
| **FavoriteDao.add() 改用 executeUpdate**    | `user_favorites` 表无自增主键（复合主键 user_id+body_id），`executeInsert` 的 RETURN_GENERATED_KEYS 始终返回 -1。改为 `executeUpdate` 检查受影响行数 |
| **DashboardPanel 未修改**                   | 交接文档标记为"检查确认"，快捷入口已正确跳转，无需修改代码                                                                                           |
| **BodyType.java 未修改**                    | 交接文档第 5 节要求添加 fromString()，实际 S1 已包含此方法，无需修改                                                                                 |
| **CelestialBodyPanel 构造器仅取 User 参数** | 交接文档写 `CelestialBodyPanel(User currentUser)`，但无 MainFrame 引用。对话框通过 `SwingUtilities.getWindowAncestor()` 获取父窗口                   |
| **ImportCsvDialog 增加 onSuccess 回调参数** | 导入成功后需刷新面板表格，增加 `Runnable onSuccess` 参数                                                                                             |
