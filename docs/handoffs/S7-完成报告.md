## 5. 核心架构决策

- **NightSkyService** 使用简化天文算法（月相 29.53 天周期、日落正弦近似、恒星时 GST+经度修正），非精密星表，误差 ±30 分钟内对推荐场景足够
- **星座 RA 数据** 硬编码 12 个主要星座在 `CONSTELLATION_RA` 静态 Map 中，后续可外置为 JSON 资源文件
- **NightSkyPanel → AddObsDialog 集成链路：** NightSkyPanel → MainFrame.openAddObservation() → ObservationPanel.showAddDialogForBody() → AddObsDialog 新构造器预填星体
- **Bortle 颜色渲染** 9 级渐变色：深蓝(1 极暗) → 绿(4-5 郊区) → 红(9 市中心)，暗色背景白字/亮色背景黑字
- **月相 Unicode 符号** 使用 `Font.SANS_SERIF` 避免 Windows 下方框问题（遵循 S3 偏差经验）
