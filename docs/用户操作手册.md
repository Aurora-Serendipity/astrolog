# AstroLog 用户操作手册

> 版本：v1.0 | 适用：AstroLog 天文观测日志与星体管理系统

---

## 1. 系统概述

AstroLog 是一款基于 Java SE 17 + Swing + MySQL 8.0 的天文观测日志与星体管理桌面应用。主要功能包括：星体数据库管理、观测记录追踪、器材管理、统计图表可视化、今夜星空推荐、梅西耶马拉松追踪、星座文化馆、报告导出。

---

## 2. 安装与启动

参见《AstroLog 安装部署指南》。

简要步骤：

1. 安装 JDK 17+、MySQL 8.0+
2. 执行 `sql/init.sql` 初始化数据库
3. 修改 `db.properties` 中的数据库连接信息
4. 双击 `astrolog.jar` 或执行 `java -jar astrolog.jar` 启动

---

## 3. 注册与登录

首次使用需注册账号：

1. 启动应用后进入登录界面
2. 点击「注册」按钮，填写用户名、密码、确认密码、城市、默认经纬度
3. 注册成功后返回登录界面，输入用户名和密码登录
4. 连续 5 次登录失败后账号将被临时锁定 30 分钟

---

## 4. 星体库浏览与管理

登录后点击左侧导航「星体库」进入星体管理面板。

**功能：**

- **浏览星体：** 表格展示全部星体（名称、类型、星座、赤经/赤纬、视星等、梅西耶编号等）
- **搜索过滤：** 按星座、类型、亮度范围、季节、关键词筛选
- **模式切换：** 全部星体 / 收藏排行 / 我的收藏
- **收藏星体：** 点击表格最后一列 ★/☆ 切换收藏状态
- **查看详情：** 双击星体行弹出详情对话框
- **管理操作（管理员）：** 添加、编辑、删除星体、导入 CSV

---

## 5. 观测记录添加与查询

点击「我的观测」进入观测记录面板。

**功能：**

- **添加观测：** 选择星体、器材、观测地 → 填写视宁度、天气、月相等 → 保存
- **查询历史：** 按日期范围、星体名称、器材筛选
- **编辑/删除：** 选中记录后操作
- **标签管理：** 为观测记录添加自定义标签

---

## 6. 器材管理与维护日志

点击「器材柜」进入器材管理面板。

**功能：**

- **器材登记：** 添加望远镜、目镜、相机等器材（名称、型号、类型、参数）
- **维护日志：** 记录器材维护/校准日期和详情
- **器材状态：** 正常 / 待维护 / 故障，一目了然

---

## 7. 统计图表与可视化

点击「统计图表」进入数据可视化面板。

**图表类型（7 个选项卡）：**

- **年度统计：** 柱状图展示各年观测频次
- **月度统计：** 柱状图展示各月观测频次
- **类型分布：** 饼图展示星体类型分布
- **年度趋势：** 折线图展示多年变化趋势
- **观测条件：** 雷达图展示视宁度、晴夜率、月相适宜度、器材多样性
- **日历热力：** 年度每日观测热度图
- **星座分布：** 星图展示 88 星座连线 + 已观测/未观测星体位置

面板底部显示器材使用排行 Top 10。数据每 15 秒自动刷新。

---

## 8. 今夜星空推荐

点击「今夜星空」进入实时天象推荐面板。

**功能：**

- 输入日期、经纬度（或从已保存的观测地选择）
- 查询结果显示：月相图标、日落时间、黄金观测窗口、月出/月落时间
- **亮星推荐：** 表格列出当晚可见亮星（名称、星座、视星等、类型）→ 选中后点击「开始观测」直接跳转记录
- **梅西耶天体：** 表格列出当晚可见梅西耶天体 → 选中后点击「开始观测」直接跳转记录

---

## 9. 梅西耶马拉松追踪

点击「梅西耶马拉松」进入 110 个梅西耶天体追踪面板。

**功能：**

- **天体目录：** 表格展示全部 110 个梅西耶天体（M#、名称、类型、星座、视星等、最佳季节、观测状态）
- **筛选：** 按类型（星云/星团/星系）、季节（春/夏/秋/冬）、状态（已观测/未观测）筛选
- **进度条：** 实时显示观测进度百分比
- **类型分类统计：** 按类型显示完成情况
- **详情查看：** 双击未观测天体弹出详情对话框（含描述）
- **跳转星体库：** 双击已观测天体 → 自动跳转到星体库并高亮对应星体
- **证书生成：** 全部 110 个观测完成后，可生成梅西耶马拉松完成证书 PDF

---

## 10. 星座文化馆

点击「星座文化馆」进入 88 星座文化展示面板。

**功能：**

- **星座列表：** 左侧列表展示全部 88 星座
- **星座详情：** 选中星座后显示缩写、面积、最亮星、最佳观测季节
- **神话传说：** 展示该星座的希腊神话与中国星官文化
- **星图查看：** 「查看星座分布图」按钮展示该星座在 88 星座星图中的位置

---

## 11. 报告导出

报告导出入口：左侧导航「系统设置」→ 点击「生成年度报告」或「梅西耶证书」。

**HTML 年度报告：**

- 包含观测摘要（总次数/星体数/器材数）、星体类型分布表、器材使用排行
- 生成后自动在预览对话框中打开
- 可导出 HTML 文件或进一步导出为 PDF

**PDF 年度报告：**

- 格式化的 JasperReports PDF 报告，含观测摘要和生成信息

**梅西耶证书：**

- 完成全部 110 个梅西耶天体观测后可生成 PDF 证书
- 证书包含观测者姓名、完成日期、认证信息

---

## 12. 系统设置

点击「系统设置」进入设置面板。

**外观设置：**

- **主题切换：** 暗色主题 / 亮色主题 / 星空主题，选中即生效，全局 UI 自动刷新
- **字体大小：** 小 / 中(默认) / 大，选中即生效

**数据管理：**

- **备份数据：** 将全部 11 张表数据导出为 SQL 文件，选择保存目录后点击「开始备份」
- **恢复数据：** 从 SQL 备份文件恢复数据（将覆盖当前数据！），请谨慎操作

**报告导出：**

- **生成年度报告：** 生成当前年份 HTML 报告并预览
- **梅西耶证书：** 完成 110/110 后生成 PDF 证书

**关于：** 展示 AstroLog 版本号和系统信息。
